# Vet clinic database

In this project, I used a relational database to create the initial data structure for a vet clinic. I created a table to store animals' information, insert some data into it, and query it.

- Major language

  - SQL

- Technologies used:
  - Git
  - GitHub
  - Vs code
  - Postgresql
